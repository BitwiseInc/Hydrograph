Count

The count function calculates the count of individual records within a group. This function requires following inputs:

  • Key fields: The key fields for cumulate. This information is already provided using the key fields option on cumulate properties window.

  • Input field name: The input field on which to count the records within the group. This information is already provided using the operation input fields option on cumulate operation editor window. Input field supports following data types:

    • Integer
    • Float
    • Double
    • Short
    • Long
    • BigInteger
    • BigDecimal

    • Output field name: The field which stores the count of records within the group. This information is already provided using the operation output fields option on cumulate operation editor window.

      Note: The data type of output field is Long.
