Sum

The sum function calculates the sum of the specified field within a group. This function requires following inputs:

  • Key fields: The key fields for aggregation. This information is already provided using the key fields option on aggregate properties window.

  • Input field name: The input field which is to be added up within the group. This information is already provided using the operation input fields option on aggregate operation editor window. Input field supports following data types:

    • Integer
    • Float
    • Double
    • Short
    • Long
    • BigInteger
    • BigDecimal

  • Output field name: The field which stores the sum of the input field within the group. This information is already provided using the operation output fields option on aggregate operation editor window.

    Note: The data type of output field is same as that of input field.
