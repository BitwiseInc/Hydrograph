Min

The min function calculates the minimum value for the specified field within the group. This function requires following inputs:

  • Key fields: The key fields for aggregation. This information is already provided using the key fields option on aggregate properties window.

  • Input field name: The input field whose minimum value within the group is to be found. This information is already provided using the operation input fields option on aggregate operation editor window.

    Note: Input field supports all comparable data types. All numerical data types as well as date, time, string etc. data types are supported. In case of string, lexicographical comparison is implemented. The comparison is based on the Unicode value of each character in the strings.

  • Output field name: The field which stores the minimum value of the input field within the group. This information is already provided using the operation output fields option on aggregate operation editor window.

    Note: The data type of output field is same as that of input field.
