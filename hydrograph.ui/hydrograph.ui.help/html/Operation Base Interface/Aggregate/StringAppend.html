StringAppend

The string append function appends all the values of the specified field within the group. The field separator can be specified using operation properties. The default field separator is ','. To specify a different field separator, add operation property with name = 'filler'. The value of this property will be used as the field separator.

  • Key fields: The key fields for aggregation. This information is already provided using the key fields option on aggregate properties window.

  • Input field name: The input field which is to be appended. This information is already provided using the operation input fields option on aggregate operation editor window.

    Note: Input field supports only string data type.

  • Output field name: The field which stores the appended string within the group. This information is already provided using the operation output fields option on aggregate operation editor window.

    Note: The data type of output field is also string.
