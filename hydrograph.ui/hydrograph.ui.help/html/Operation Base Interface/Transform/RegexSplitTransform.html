RegexSplitTransform

The RegexSplitTransform function splits the input records based on the regex to create multiple records on the out port. This function accepts one input and one output field. The input field is split based on the regex provided to the RegexSplitTransform function. The output field holds the values split based on the regex.

This function requires following inputs:

  • Input fields: The field to split the input on. RegexSplitTransform accepts just 1 input field on which the regex is applied. The input filed is passed to the RegexSplitTransform function using input fields grid on transform operation editor window.
  • Output fields: The field present on the output. RegexSplitTransform accepts just 1 output field which holds the values split based on regex. These are passed to the RegexSplitTransform function using output fields grid on transform operation editor window.
  • regex: The regular expression to apply on the input field to split the data into multiple records. The regex is passed to RegexSplitTransform function using operation properties grid on operation class window. The name of the property should be regex and the value of the property should be the actual regex to be applied on the input field.
