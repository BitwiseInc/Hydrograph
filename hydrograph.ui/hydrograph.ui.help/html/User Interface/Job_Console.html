Job Console

Job Console acts as an interface that displays execution results of the job that is curently running.

It is used to monitor the execution of a job step-by-step. Along with the timestamp of job execution and the user's system details are also displayed in the console.

Limiting Console Output

Console buffer size limits the number of characters buffered in the console. A maximum buffer size must also be specified. When console output surpasses the specified maximum, output is truncated from the beginning of the buffer. Default value for Hydrograph console buffer is 10000 characters. Max value for Hydrograph console log is 40000.
