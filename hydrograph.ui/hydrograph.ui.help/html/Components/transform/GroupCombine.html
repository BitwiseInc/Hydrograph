GroupCombine

GroupCombine component provides group combine operations on the input. It is present under Transform category in the component palette. combinations are performed on groups of data as defined by the user. It contains one input port and one output port.

‘GroupCombine’ created to enhance the performance of Aggregate component. Working of GroupCombine component would be similar to Aggregate component with few additional configurations

User is provided with options to either create a custom Java class that specifies the GroupCombine operation or select a pre-defined standard Operation Class.

The GroupCombine component can be easily distinguished in the component palette.

An enlarged version of the same is depicted when dragged on the canvas.
