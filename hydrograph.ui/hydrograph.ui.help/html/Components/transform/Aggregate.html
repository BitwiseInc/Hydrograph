Aggregate

Aggregate component provides aggregate operations on the input. It is present under Transform category in the component palette. Aggregations are performed on groups of data as defined by the user. It contains one input port and one output port.

User is provided with options to either create a custom Java class that specifies the aggregate operation or select a pre-defined standard Operation Class.

The Aggregate component can be easily distinguished in the component palette.

An enlarged version of the same is depicted when dragged on the canvas.
