Output Hive Parquet

Output Hive Parquet as the name suggests accepts a parquet file as input source and is present under the Output category in the component palette.

This component is used to run a Hive Job and store ouput as Parquet file. It accepts the hive database name and table name from the user in the component properties and has a single input port.

The Output Hive Parquet component can be easily distinguished in the component palette.

An enlarged version of the same is depicted when dragged on the canvas.
