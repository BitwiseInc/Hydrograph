Output File Mixed Scheme

Output File Mixed Scheme as the name suggests writes a mixed scheme file as output and is present under the Output category in the component palette.

This component is used to write a mixed scheme text file. It accepts the delimiter or length of field from the user in the component schema properties and has a single output port.

The Output File Mixed Scheme component can be easily distinguished in the component palette.

An enlarged version of the same is depicted when dragged on the canvas.
