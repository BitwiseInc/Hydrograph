Output Teradata

Output Teradata as the name suggests will load records from job into Teradata database. It is present under the Output category in the component palette.

Teradata Output Component will load records from job into Teradata database letting you to insert directly to required destination table or through an SQL statements. The records can be new inserts or update or replace to the target table.

The Output Teradata component can be easily distinguished in the component palette.

An enlarged version of the same is depicted when dragged on the canvas.
