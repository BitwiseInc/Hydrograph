Output Oracle

Output Oracle as the name suggests write records in Oracle table as output. It is present under the Output category in the component palette.

Oracle Output Component will load data to Oracle database in required destination table. The records can be new inserts or update or replace to the target table.

The Output Oracle component can be easily distinguished in the component palette.

An enlarged version of the same is depicted when dragged on the canvas.
