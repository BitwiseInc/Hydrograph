Input File Mixed Scheme

Input File Mixed Scheme as the name suggests accepts a mixed scheme file as input source and is present under the Input category in the component palette. Mixed scheme file can contain fixed length fields and can also vary in delimiter.

This component is used to read mixed scheme text file. It accepts the delimiter from the user in the component properties and has a single output port.

The Input File Mixed Scheme component can be easily distinguished in the component palette.

An enlarged version of the same is depicted when dragged on the canvas.
