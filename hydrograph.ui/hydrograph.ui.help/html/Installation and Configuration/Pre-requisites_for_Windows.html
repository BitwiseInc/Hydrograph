Pre-requisites for Windows

 Setup Requirements on developer machine

  •  System requirements

    •  Windows 7 and above (64 bit)

    •  8 GB RAM(minimum, 2 GB is required by Hydrograph IDE)

  •  Software requirements

    •  JDK 1.8, 64bit

    •  Gradle 3.0(minimum, used to build the jobs and run it locally or remotely. Download

    •  Hadoop, 64bit(required for running jobs locally)

  After installing the required software, configure following values in System Environment variables:

  •  JAVA_HOME=< jdk 1.8 home directory path >

  •  HADOOP_HOME=< hadoop installation directory path >

  •  PATH=%PATH%;%JAVA_HOME%\bin; < gradle installation directory >\bin;%HADOOP_HOME%\bin;

  Make sure that GRADLE and HADOOP bin directories have execute permission
